1. Overview

Amazon's Product Advertising API (PAAPI) 5.0 allows developers to access Amazon's product catalog, search for items, retrieve product details, and generate affiliate links. This API uses JSON format over HTTPS and offers SDK support for popular programming languages.

PAAPI 5.0 replaced the older PAAPI 4.0 version with improvements in response times, data structure, and features like multilingual support and category-specific sales rank information.
2. API Operations
Operation Description Endpoint Required Parameters
SearchItems Search for products on Amazon based on keywords, category, etc. /paapi5/searchitems PartnerTag PartnerType At least one search parameter
GetItems Retrieve detailed information about specific items by their ASINs /paapi5/getitems PartnerTag PartnerType ItemIds
GetVariations Get information about product variations (e.g., different sizes/colors) /paapi5/getvariations PartnerTag PartnerType ASIN
GetBrowseNodes Retrieve details about Amazon's category hierarchy /paapi5/getbrowsenodes PartnerTag PartnerType BrowseNodeIds
3. Common Request Parameters
Parameter Description Required Format/Example
AccessKey Your Amazon PAAPI access key Yes AKIAIOSFODNN7EXAMPLE
SecretKey Your Amazon PAAPI secret key Yes wJalrXUtnFEMI/K7MDENG/bPxRfiCYEXAMPLEKEY
PartnerTag Your Associate/affiliate tag Yes yourtag-20
PartnerType Type of partner program Yes Associates
Marketplace Target Amazon marketplace No www.amazon.com
LanguagesOfPreference Preferred language for responses No ["en_US"]
CurrencyOfPreference Preferred currency for price display No USD
The PartnerTag must be valid for the marketplace you're targeting. For example, a US tag (yourtag-20) won't work with the UK marketplace.

5. SearchItems Operation

The SearchItems operation searches for products on Amazon based on various criteria.

Parameter Description Example
Keywords Search terms "wireless headphones"
SearchIndex Category to search in "Electronics"
BrowseNodeId Specific category node ID "172282"
Author Book author name "J.K. Rowling"
Brand Product brand name "Sony"
MinReviewsRating Minimum star rating "4"
MinPrice/MaxPrice Price range filters "10"
Merchant Merchant type "Amazon"
ItemCount Number of items to return (max 10) "10"
ItemPage Page number for pagination "1"
SortBy Result sorting order "Price:HighToLow"
At least one search parameter (Keywords, Actor, Artist, Author, Brand, Title) is required.
6. GetItems Operation

The GetItems operation retrieves detailed information about specific products by their ASINs.

Parameter Description Example
ItemIds List of ASINs to retrieve ["B07JQKGZ9S", "B01N5IB20Q"]
ItemIdType Type of ID (default is ASIN) "ASIN"
Condition Product condition "New"
Merchant Merchant type "All"
Resources Requested information ["Images.Primary.Large", "ItemInfo.Title"]
7. GetVariations Operation

The GetVariations operation retrieves information about a product's variations (e.g., different sizes/colors).

Parameter Description Example
ASIN Parent ASIN of the variation family "B07H65KP63"
Condition Product condition "New"
Merchant Merchant type "All"
VariationCount Number of variations to return "10"
VariationPage Page number for pagination "1"
Resources Requested information ["VariationSummary.VariationDimension"]
8. GetBrowseNodes Operation

The GetBrowseNodes operation retrieves information about Amazon's category hierarchy.

Parameter Description Example
BrowseNodeIds List of browse node IDs ["283155", "3040"]
LanguagesOfPreference Preferred language for responses ["en_US"]
Resources Requested information ["BrowseNodes.Ancestor", "BrowseNodes.Children"]
9. Rate Limits and Throttling

Amazon imposes rate limits on PAAPI 5.0 requests. Initial limits for new accounts:

  • 1 request per second (TPS)
  • 8,640 requests per day (TPD)

Rate limits are adjusted based on sales performance:

  • +1 TPD for every 5 cents in shipped item revenue
  • +1 TPS (up to max 10) for every $4,320 in shipped item revenue
If your account doesn't generate sales via PAAPI links for 30 consecutive days, you may lose API access.
10. Error Handling
Error Type HTTP Code Description Retry?
TooManyRequestsException 429 Rate limit exceeded Yes, with backoff
ValidationException 400 Invalid parameters No
MissingParameter 400 Required parameter not provided No
InvalidPartnerTag 400 Partner tag not valid for account No
InvalidSignatureException 400 Request not signed correctly No
UnknownOperationException 400 Invalid operation No
ItemNotAccessible 200 (partial success) Item cannot be accessed via API No
ServiceUnavailable 503 Temporary service issue Yes, with backoff
For retryable errors, implement exponential backoff with jitter to distribute retry attempts and avoid request spikes.
11. SDK Support

Amazon provides official SDKs for several programming languages:

Language Installation Notes
Java Download from Amazon documentation Includes dependencies folder with required JARs
PHP Download from Amazon documentation Requires PHP 7.0+
Python Download from Amazon documentation Requires Python 3.6+
Node.js npm install amazon-paapi Unofficial wrapper for Amazon's SDK
Several community-maintained wrappers also exist for various programming languages.

Example SDK Usage (Node.js):

const amazonPaapi = require('amazon-paapi');

const commonParameters = {
  AccessKey: 'YOUR_ACCESS_KEY',
  SecretKey: 'YOUR_SECRET_KEY',
  PartnerTag: 'your-tag-20',
  PartnerType: 'Associates',
  Marketplace: 'www.amazon.com'
};

const requestParameters = {
  Keywords: 'harry potter books',
  SearchIndex: 'Books',
  ItemCount: 10
};

amazonPaapi.SearchItems(commonParameters, requestParameters)
  .then(response => {
    console.log('API response:', response);
  })
  .catch(error => {
    console.error('Error calling API:', error);
  });
12. Troubleshooting

Common issues and solutions when working with the Product Advertising API.

Issue Possible Cause Solution
Authentication failed Incorrect access key or secret key Verify your credentials in the Amazon Associates account
Request throttled Exceeding rate limits Implement exponential backoff retry logic
Invalid parameter value Using incorrect parameter format Check the API documentation for correct parameter formats
No results found Search criteria too restrictive Broaden search terms or try different parameters
Missing required resources Not specifying necessary resources Include all required resources in your request
If you're consistently getting errors, check the official troubleshooting guide or contact Amazon support.
